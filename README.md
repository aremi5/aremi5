## 🔬 Labs

### ✅ Enterprise Active Directory Lab — RemiTech IT Environment
> **Azure | Windows Server 2022 | Windows 11 | PowerShell | ServiceNow**

A fully functional enterprise domain environment built from scratch in Microsoft Azure. Covers the complete lifecycle of domain administration — from infrastructure deployment to security hardening, network diagnostics, and ITSM ticketing.

**Scenarios completed:**
- Azure VM deployment (DC01 + CLIENT01) in shared VNet
- AD DS installation, domain promotion, OU structure design
- Bulk user provisioning via PowerShell (10 users, 5 departments)
- Security groups and RBAC using the AGDLP model
- Group Policy — password policy, lockout policy, security baseline
- Print Server role + GPO-based printer deployment to CLIENT01
- SMB file share with NTFS and share-level group permissions
- Account lockout simulation, Event Viewer investigation (Event ID 4740, 4625)
- TCP/IP troubleshooting — ipconfig, ping, nslookup, netstat, flushdns
- Wireshark captures — DNS, SMB2, Kerberos
- RDP failure simulation, diagnosis, and recovery via Azure Run Command
- ServiceNow incident lifecycle — created, work notes, resolved
